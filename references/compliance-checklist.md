# 日生研新媒体内容合规检查清单

> 本清单适用于所有公众号/小红书/抖音内容发布前的合规审查。
> 每篇内容发布前必须逐项检查，任何一项"❌"都必须修改后才能发布。

---

## 一、选题阶段合规预检（写作前必查）

### 1.1 选题红线检查

| # | 检查项 | 通过标准 | 结果 |
|---|--------|----------|------|
| 1 | 选题是否暗示纳豆激酶可治疗/治愈疾病？ | 不得出现"治疗""治愈""根治"等医疗承诺 | ☐ |
| 2 | 选题是否涉及具体疾病诊断？ | 不得替代医生给出诊断建议 | ☐ |
| 3 | 选题是否与竞品做疗效对比？ | 不得贬低同行或做功效对比 | ☐ |
| 4 | 选题角度是否可追溯到知识库素材？ | 所有核心观点必须有知识库支撑 | ☐ |
| 5 | 选题是否符合"食品"定位？ | 全文语境必须在食品范畴内 | ☐ |

### 1.2 选题安全分级

| 级别 | 定义 | 处理方式 |
|------|------|----------|
| 🟢 安全 | 品牌故事/行业科普/大会报道/用户教育 | 直接写作 |
| 🟡 谨慎 | 涉及临床数据/健康功效/专家观点 | 写作时逐句合规审查 |
| 🔴 高危 | 涉及疾病名称/治疗效果/医学建议 | 必须专人审核，建议法务确认 |

## 四、禁用词速查表

### 4.1 绝对禁止（出现即违法）

| 禁用词 | 安全替代 |
|--------|----------|
| 治疗/治愈/根治 | "辅助健康管理""营养补充" |
| 药物/药品/处方 | "功能性食品""营养食品" |
| 疗效/药效 | "研究数据显示""临床观察发现" |
| 预防XX病/防治 | "有助于维护XX健康" |
| 最好/最强/第一/唯一 | "领先""优质""代表性" |
| 100%有效/保证见效 | "研究显示XX%的受试者..." |
| 无副作用/绝对安全 | "安全性数据良好""耐受性好" |

### 4.2 高风险词（平台易触发审核）

| 高风险词 | 安全替代 |
|----------|----------|
| 降血压/降血脂/降血糖 | "有助于维护心血管健康" |
| 溶栓/溶解血栓 | "纤溶酶活性研究" |
| 抗癌/防癌 | 禁止使用，不做替换 |
| 消炎/杀菌 | "有助于维护身体平衡" |
| 排毒/清肠 | "膳食营养补充" |
| 增强免疫力 | "营养支持" |
| 延缓衰老/抗衰 | "营养健康管理" |
| 患者/病人 | "关注健康的人群""受试者" |
| 临床验证/医学证明 | "研究数据显示""科学观察" |
| 脉管焕新 | 禁止使用（同行已因此被央视点名） |

### 4.3 AI味/营销号词汇

赋能、链路、飞轮、认知升级、颗粒度、抓手、闭环、触达、矩阵、范式、底层逻辑（单独使用）、护城河（泛化使用）、降维打击、生态位、心智模型、放大招、炸裂、颠覆

### 4.4 禁用句式

「不是…而是…」「不是…是…」「并非…而是…」→ 改用直接陈述或"说白了""本质上"引出判断

### 4.5 平台专属敏感词

**小红书额外禁用**：
- "私聊""加微""详询客服" → 用"评论区留言"
- "代理""加盟""拿货" → 用"合作伙伴计划"
- "月入XX万""躺赚" → 禁止使用

**抖音额外禁用**：
- "点击购物车""橱窗" → 食品类需单独审核资质
- "医生推荐""专家推荐" → 需实名认证资质

**公众号额外注意**：
- 外部链接（淘宝/京东） → 可能被判定诱导外链
- "限时优惠""最后XX名" → 可能触发营销类限流

## 五、安全写作模板

### 模板A：引用临床数据（最常用）

```
✅ 安全写法：
"在一项由[医院名称][科室]开展的研究中，[XX]名受试者参与了为期[XX]周的观察。
研究数据显示，[具体指标]呈现[具体变化趋势]。
该研究已发表于[期刊名称]，供学术参考。

温馨提示：本品为食品，不能替代药物。如有健康问题，请咨询专业医生。"

❌ 违规写法：
"临床验证，纳豆激酶能有效降血压，1062名患者证明100%有效！"
```

### 模板B：专家观点引用

```
✅ 安全写法：
"[专家姓名]（[完整职务]）在[具体场合，如FFC 2026大会]上指出：
'[直接引用原话]'
以上为专家个人学术观点，仅供参考。"

❌ 违规写法：
"权威专家证明纳豆激酶能治病！浙二医院主任医师都在推荐！"
```

### 模板C：品牌故事/招商软文

```
✅ 安全写法：
"日生研深耕纳豆激酶领域15年，坚持[具体标准]。
目前已与全国[XX]家健康服务机构建立合作。
如果您对健康食品事业感兴趣，欢迎了解我们的合作伙伴计划。"

❌ 违规写法：
"加盟日生研月入10万！纳豆激酶治百病，躺赚！"
```

## 七、法规依据索引

| 法规/规范 | 核心条款 | 与日生研的关系 |
|-----------|----------|----------------|
| 《广告法》第17条 | 除医疗、药品、医疗器械外，不得涉及疾病治疗 | 纳豆激酶是食品，不得宣传治疗功效 |
| 《广告法》第18条 | 保健食品广告限制 | 日生研产品是普通食品，更不能碰保健食品的边 |
| 《广告法》第9条 | 禁止"最高级""国家级"等绝对化用语 | 所有宣传材料必须避免 |
| 《广告法》第4条 | 广告不得含有虚假或引人误解的内容 | 所有数据引用必须真实可查 |
| 《广告法》第11条 | 使用数据、统计资料等应真实准确，标明出处 | 引用临床数据时必须标注来源 |
| 《食品安全法》第71条 | 食品标签/说明不得涉及疾病预防治疗 | 所有产品描述必须遵守 |
| 《食品安全法》第73条 | 食品广告不得涉及疾病预防治疗 | 所有广告内容必须遵守 |
| 《食品安全法实施条例》第38条 | 禁止利用学术论文等为食品做商业宣传 | 引用临床数据时必须客观中立 |
| 四部门《关于规范自媒体医疗科普行为》 | 非医疗资质账号不得发布医疗科普 | 日生研账号发布健康内容需谨慎定位 |
| 微信公众平台运营规范 | 健康类内容审核、外链管控 | 公众号发布必须遵守 |
| 小红书《广告违禁词规范》 | 绝对化用语、功效承诺禁止 | 小红书发布必须遵守 |
| 抖音《健康内容管理规范》 | 健康科普创作者资质要求 | 抖音发布需确认账号资质 |

## 八、处罚案例警示（前车之鉴）

### 案例1：纳豆激酶胶囊虚假宣传，罚款40万
- **违规行为**：宣传纳豆激酶胶囊具有"溶栓""降血压"等药品功效
- **处罚依据**：《广告法》第17条、第55条
- **教训**：普通食品宣传药品功效，罚款20万起步

### 案例2：三生研直播间被央视点名
- **违规行为**：直播中使用"脉管焕新"等暗示性表述
- **教训**：即使不直接说"治疗"，隐晦暗示同样违规。**同赛道前车之鉴，必须重视**

### 案例3：氢养馆虚假宣传立案
- **违规行为**：宣传"氢气吸入"可治疗多种疾病
- **教训**：健康类产品夸大功效，市场监管局立案查处

### 案例4：2026年3月专项整治
- **背景**：市场监管总局联合多部门开展食品广告专项整治（2026年3月-9月）
- **重点**：功能性食品、保健食品虚假宣传
- **影响**：当前正处于执法高压期，所有内容必须更加谨慎
