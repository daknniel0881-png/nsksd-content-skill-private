# 科普大白话写作风格(日生研公众号专用)

> 这是给日生研公众号撰稿官唯一的风格指南。
> **不要**引用任何个人写作风格(包括 quyu-writing-style)。
> 目标读者:40-60 岁的三高人群 + 美容/养生/社区门店老板。

---

## 一、核心风格定位

**像社区医生跟邻居唠嗑。**

- 不是论文,不是广告,不是鸡汤
- 不讲概念,讲场景
- 不讲大道理,讲具体事
- 不说"服用后",说"吃下去之后"
- 不说"研究表明",说"杭州市一97例临床研究"或"浙大120人RCT"（v8.4：弱化日本表述；V10.6：禁用1062人竞品数据）

参考频道(不是模仿,是体会风格)：
- 丁香医生公众号
- 中科院物理所公众号(科普部分)
- 果壳网早期风格
- 段子手之后的"回形针 PaperClip"

## 二、硬约束(违反直接改)

### 段落与小标题（V9.9 硬约束）

- **单段 ≤ 100 字**。超过必拆成两段
- **每篇必须有 3-6 个二级小标题（##）**，分章节组织内容
- 小标题本身是完整观点，不写"背景/方法/结论"式目录
- 换角度即起新小标题，不要一大段到底
- 段落之间留空行，不靠 `----` 撑节奏

### 字数
- 公众号文章:**1500-2500 字**
- 低于 1500 读不过瘾,高于 2500 完读率掉
- 段落:每段 ≤ 5 行,长段必拆

### 禁用词(一眼 AI)

**商业黑话**:赋能、链路、飞轮、颗粒度、抓手、闭环、触达、矩阵、范式、护城河、底层逻辑、降维打击、生态位、心智模型、跃迁、复利

**AI 客套**:综上所述、值得一提的是、无独有偶、不难发现、可以想见、众所周知、毋庸置疑、可见一斑、由此可见、总而言之

**模糊废话**:很多、大部分、各种各样、若干、一定程度上、相关人士、有关专家、业内人士

发现即改。具体替换:"很多人"→"80% 的人";"有关专家"→"EFSA 食品安全局";"一定程度上"→"降幅在 15-20%"。

### 禁用句式

- ❌ "不是 X,而是 Y" / "并非 X,而是 Y"
- ❌ "不仅 X,而且 Y"
- ❌ "一方面 X,另一方面 Y"
- ❌ "首先 X,其次 Y,最后 Z"(当连续出现时特别 AI)

改法:拆成 2-3 句短句,或直接陈述。

### 破折号规则
- 整篇 `——` 或 `—` 总数 **≤ 2 个**
- 优先用逗号或拆句
- 不是"更地道"就要多用

### 感叹号
- 整篇 `!` 总数 **≤ 3 个**
- 绝大多数用句号或问号
- 问号比感叹号更能让读者停下来思考

## 三、写作技法

### 1. 钩子开场(50-80 字)

**不要**从"随着生活水平提高""当今社会"开头。

**要**从具体场景切入:
- "58 岁的李阿姨上周体检,血压 165/98。"
- "隔壁张哥最近老说胸口发闷,去年体检甘油三酯就超了。"
- "这周接到一个美容院老板电话,说她们店要开始卖纳豆激酶。"

### 2. 具体 > 抽象

| 抽象(差) | 具体(好) |
|---------|---------|
| 很多研究证明有效 | 杭州市一97例临床研究,连续服用90天后 |
| 欧洲认证 | EFSA(欧盟食品安全局)2011 年通过的健康声称 |
| 能提高品质 | 每 1 克含 20,000 FU 酶活性(普通纳豆激酶约 2,000 FU) |
| 很多美容院 | 上海静安某街商圈,近半年 6 家美容院上了这款 |

### 3. 数据必注出处

每个数字后必须附:
- `(来源:EFSA 2011 健康声称评估)`
- `(数据:市一97例临床/浙大120人RCT,《中华养生保健》第42卷第5期/《卒中与脑血管疾病杂志》)`
- `(根据浙大 2019 年发表的研究)`

没出处的数字一律删掉,不要编。

### 4. 类比解释难词

- "纤溶酶" → "就是身体里专门拆血栓的那把剪刀"
- "LDL-C 低密度脂蛋白" → "俗称'坏胆固醇',血管里的油垢主力"
- "血浆黏度" → "简单说就是血稠不稠"

### 5. 节奏:短中长句交替

**全是长句(差)**:
> 纳豆激酶作为一种由纳豆菌发酵大豆产生的丝氨酸蛋白酶,在临床研究中展现出显著的纤溶活性,对血栓的预防和溶解具有一定的辅助作用,因此在心脑血管疾病的辅助干预中受到广泛关注。

**短中长交替(好)**:
> 纳豆激酶从哪来?简单说,就是纳豆发酵的时候菌种搞出来的一种酶。它的本事,是能帮身体拆血栓——这话不是我说的,杭州市一97例临床,连续吃90天,斑块改善率46.7%(数据来源:《中华养生保健》第42卷第5期)。所以对三高、老人血管这种场景,它是个辅助选项。

## 四、结构化表达

用编号、短横线列表、对比表格,**但别过度**。

- 每篇 ≤ 2 个列表
- 每个列表 ≤ 5 项
- 表格只在对比时用(A vs B,前 vs 后)

## 五、结尾写法

**不要**催下单、不要喊口号、不要"关注我们"。

**要**:
- 开放式问句:"你身边有这样的人吗?"
- 留白:"关于纳豆激酶,我们下次再聊聊它不适合的人群。"
- 具体动作:"想了解分销的,后台回复'分销'两个字。"

## 六、6 段式标准结构

按这个顺序写(大纲官会给好):

1. **钩子**(50-80 字)具体场景
2. **问题陈述**(200-300 字)谁遇到了什么痛点
3. **科学证据**(300-500 字)数据 + 出处,拆血栓机制讲清
4. **产品衔接**(200-300 字)NSKSD 的具体差异点(FU 值、来源、工艺)
5. **赚钱逻辑**(200-400 字,招商向才写)分销模式、利润空间、客户画像
6. **收尾**(50-100 字)开放式

## 七、自查清单(写完过一遍)

- [ ] 字数 1500-2500?
- [ ] AI 黑话 0 命中?
- [ ] 破折号 ≤ 2?
- [ ] 感叹号 ≤ 3?
- [ ] 每个数据都有出处?
- [ ] 开场是具体场景,不是"当今社会"?
- [ ] 有 3 个以上具体数字?
- [ ] 没用"不是 X 而是 Y"句式?
- [ ] 结尾是问号而不是感叹号?

命中任一项"否",回头改。

## 八、日本表述弱化（v8.4 新增 · 硬约束）

国际关系敏感期，所有对外内容禁止出现 `日本/日式/日系/东瀛/和风/中日` 等关键词。必须使用：
- "原研方 / 国际合作研发机构"替代"日本生物科学研究所"
- "国际原料 / 进口原料"替代"日本进口"
- "一项 XX 人的临床研究 / 国际研究"替代"日本做过的研究"
- "NSKSD纳豆激酶 / 原研级纳豆激酶"替代"日本纳豆激酶"

叙事切换：从"日本来的"改为"科学循证的"；从"异域光环"改为"市一97例临床 + 浙大120人RCT + 中国专家共识"的科学主场叙事。

豁免：产品包装、法律标签、学术论文原引。
