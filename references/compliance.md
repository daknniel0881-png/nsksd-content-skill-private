# 日生研NSKSD · 完整合规审查机制

> 纳豆激酶是食品/膳食补充剂，不是药品。所有内容必须在法律合规框架内。
> 本文档包含法律法规、平台规则、处罚案例、禁用词替换表的完整合规体系。

## 第一章 法律法规层（国家级红线）

### 1.1 《中华人民共和国广告法》

**第9条**：广告不得使用"国家级""最高级""最佳"等绝对化用语。
- 与日生研的关系：所有宣传材料中禁止使用"最好的纳豆激酶""第一品牌""唯一有效"等表述

**第17条**：除医疗、药品、医疗器械广告外，任何广告不得涉及疾病治疗功能，并不得使用医疗用语或者易使推销的商品与药品、医疗器械相混淆的用语。
- 与日生研的关系：纳豆激酶是食品，绝对不能使用"治疗""治愈""预防疾病""药效"等表述

**第18条**：保健食品广告不得利用广告代言人作推荐、证明；不得涉及疾病预防、治疗功能。
- 与日生研的关系：NSKSD是普通食品（连保健食品都不是），比保健食品的限制更严格

**第4条**：广告不得含有虚假或者引人误解的内容，不得欺骗、误导消费者。
- 与日生研的关系：所有临床数据引用必须真实准确，不得虚构或篡改

**第11条**：广告使用数据、统计资料、调查结果、文摘、引用语等引证内容的，应当真实、准确，并标明出处。
- 与日生研的关系：引用1062人临床数据、浙大RCT结果时必须标注来源

### 1.2 《中华人民共和国食品安全法》

**第71条**：食品和食品添加剂的标签、说明书，不得含有虚假内容，不得涉及疾病预防、治疗功能。
- 与日生研的关系：产品描述和所有宣传内容不得暗示疾病预防治疗功能

**第73条**：食品广告的内容应当真实合法，不得含有虚假内容，不得涉及疾病预防、治疗功能。
- 与日生研的关系：所有广告内容（含自媒体文章）必须遵守

### 1.3 《食品安全法实施条例》

**第38条**：对保健食品之外的其他食品，不得声称具有保健功能；对添加食品安全国家标准规定的选择性添加物质的婴幼儿配方食品，不得以选择性添加物质命名。任何单位和个人不得利用广告或者其他方式，对食品作出不真实的宣传，不得利用国家机关、社会团体、科研机构的名义或者以学术论文等形式为食品进行宣传。
- 与日生研的关系：引用临床数据时必须客观中立，不能以"学术论文证明"的名义做商业宣传

### 1.4 四部门《关于规范自媒体医疗科普行为的通知》（2025年）

**核心要求**：
- 非医疗资质的自媒体账号不得发布医疗科普内容
- 健康类内容必须标注信息来源
- 不得以科普为名行商业推广之实
- 与日生研的关系：日生研账号不是医疗机构，发布健康内容需定位为"行业资讯/企业动态"，不能定位为"医疗科普"

### 1.5 市场监管总局2026年3月专项整治

**背景**：2026年3月起，市场监管总局联合多部门开展为期半年（至9月）的食品广告专项整治行动。
**重点查处**：功能性食品、保健食品虚假宣传
**影响**：当前正处于执法高压期，所有内容必须更加谨慎

## 第二章 平台规则层（各平台专项规定）

### 2.1 微信公众平台

**运营规范4.1 健康类内容**：
- 发布涉及疾病、症状、治疗方案的内容需要相关资质
- 不得发布未经证实的偏方、秘方
- 不得发布引导用户自行诊断、用药的内容

**运营规范3.3 外链管控**：
- 不得在文章中放置引导外部交易的链接（淘宝/京东等）
- 不得通过二维码引导到外部购买页面
- 可通过公众号菜单栏引导

**处罚机制**：
- 首次违规：内容删除 + 警告
- 多次违规：限制群发功能 + 账号降权
- 严重违规：永久封禁

### 2.2 小红书

**《广告违禁词规范》**：
- 禁止使用绝对化用语（最/第一/唯一等）
- 禁止使用夸大功效用语（根治/特效/立竿见影等）
- 禁止使用医疗术语进行非医疗产品宣传

**内容审核规范**：
- 健康类内容需标注"非医疗建议"
- 涉及功效的内容可能被限流或不予推荐
- "医疗健康"类目发布需认证资质

**平台专属敏感词**：
- "私聊""加微""详询客服" → 用"评论区留言"
- "代理""加盟""拿货" → 用"合作伙伴计划"
- "月入XX万""躺赚" → 禁止使用

**处罚机制**：
- 违规内容：限流 → 删除 → 账号降权
- 严重违规：笔记全量限流 + 搜索屏蔽
- 多次违规：账号封禁

### 2.3 抖音

**《健康内容管理规范》**：
- 健康科普类内容创作者需进行资质认证
- 未经认证的账号发布健康内容可能被限流
- 涉及具体产品的健康类内容需标注"广告"

**广告审核规范**：
- 食品类广告不得涉及疾病预防治疗功能
- "保健食品"相关内容需提供批文号
- 普通食品不得使用保健食品的宣传用语

**平台专属敏感词**：
- "点击购物车""橱窗" → 食品类需单独审核资质
- "医生推荐""专家推荐" → 需实名认证资质
- "临床证明""医学验证" → 需提供资质证明

**处罚机制**：
- 违规内容：限流 → 删除
- 涉及虚假宣传：账号扣分 → 封禁
- 涉及法律问题：配合监管部门处理

## 第三章 处罚案例（前车之鉴）

### 案例1：纳豆激酶胶囊虚假宣传，罚款40万元
- **时间**：2024年
- **违规行为**：某企业在电商平台宣传纳豆激酶胶囊具有"溶栓""降血压""清理血管"等药品功效
- **处罚结果**：市场监管局依据《广告法》第17条、第55条，罚款40万元
- **处罚依据**：普通食品宣传药品功效，属于虚假广告
- **对日生研的教训**：即使产品确实有临床数据支持，宣传用语也必须停留在"食品"范畴

### 案例2：三生研直播间被央视点名
- **违规行为**：在直播中使用"脉管焕新"等暗示性表述描述产品功效
- **后果**：品牌形象严重受损，被监管部门约谈
- **对日生研的教训**：同赛道企业的前车之鉴。即使不直接说"治疗"，隐晦暗示同样违规。**"脉管焕新"必须列入禁用词**

### 案例3：氢养馆虚假宣传立案查处
- **违规行为**：宣传"氢气吸入"可治疗多种疾病，包括癌症、糖尿病等
- **处罚结果**：市场监管局立案查处，责令整改
- **对日生研的教训**：健康类产品夸大功效的后果不只是罚款，还有品牌信誉损失

### 案例4：某保健品企业被罚150万
- **违规行为**：利用"专家讲座"形式向老年人推销保健品，夸大产品功效
- **处罚结果**：罚款150万元，吊销营业执照
- **对日生研的教训**：利用专家身份做商业推广是高风险行为，引用专家观点必须客观中立

## 第五章 数据引用准确性规范

### 5.1 常见错误 vs 正确表述

| 常见错误 | 正确表述 | 为什么 |
|----------|----------|--------|
| "纳豆激酶能溶解血栓" | "研究显示NSKSD纳豆激酶具有一定的纤溶活性" | 不能泛化为所有纳豆激酶 |
| "吃纳豆激酶斑块就消了" | "1062人临床试验显示颈动脉斑块改善率66.5%-95.4%" | 必须引用具体数据和条件 |
| "纳豆激酶治老年痴呆" | "浙大RCT研究显示NSKSD纳豆激酶使视觉空间功能衰退风险降低约65%" | 不是治疗，是降低风险 |
| "专家说纳豆激酶很好" | "北京神经内科学会等3家学术机构的30余位专家发表的《专家建议》中指出……" | 必须具体到机构和文件 |
| "央视报道了" | "人民日报客户端/新华社客户端报道了……" | 必须准确到具体媒体 |

### 5.2 NSKSD vs 纳豆激酶品类

**关键区分**：知识库中的临床数据都是针对NSKSD产品做的，不能推广到整个纳豆激酶品类。

- ❌ "纳豆激酶降低认知衰退风险65%"
- ✅ "NSKSD纳豆激酶在浙大RCT研究中显示，视觉空间功能衰退风险降低约65%"

### 5.3 争议点的坦诚处理

以下争议点不回避，但要用正确方式表述：

- **EFSA质疑口服吸收**：坦诚提及，同时指出"山东大学研究和人体药代动力学试验支持口服有效"
- **不能替代药物**：这是底线，也是专业度的体现。主动声明比被质疑好
- **不是所有人都有效**：1062人试验中不良反应自报率1.1%，意味着绝大多数人耐受良好，但不能说"100%安全"

## 第五B章 数据真实性验证（5步验证法）

> 所有文案中引用的数据、研究、专家观点，必须经过以下5步验证，确保真实性。AI生成的内容极易出现"幻觉"（编造看似真实的数据），本验证法是防止数据失真的最后防线。

### 步骤一：溯源（Trace Back）
**操作**：每一个数据点、研究结论、专家观点，都必须追溯到知识库原文。
- 找到对应的原始文件（`knowledge/核心文档/` 或 `knowledge/2025新闻/`）
- 确认数据在原文中的具体位置（哪份文件、哪个段落）
- 如果在知识库中找不到来源 → **该数据不得使用**

**输出**：为每个数据点标注`[来源：文件名 > 段落关键词]`

### 步骤二：精确核对（Exact Match）
**操作**：将文案中的数据与知识库原文逐字比对。
- 数字是否准确？（1062人不能写成1000+人）
- 百分比是否准确？（66.5%-95.4%不能写成"超过90%"）
- 时间/机构是否准确？（浙大不能写成"某大学"）
- 研究条件是否完整？（10800FU/天持续12个月，不能省略剂量或时长）

**常见错误**：AI会"四舍五入"或"概括性表述"，把精确数据变成模糊描述

### 步骤三：交叉验证（Cross Check）
**操作**：同一数据点，在知识库中寻找至少2个独立来源互相印证。
- 临床数据：核心文档 × 新闻报道 × 科学循证文献
- 专家观点：专家建议原文 × 新闻报道引用
- 企业信息：企业介绍 × 百问百答 × 新闻报道

**如果两个来源的数据有出入** → 以核心文档（科学循证文献合辑/临床册子）为准，在文案中使用更保守的表述

### 步骤四：列出来源链接（Source Listing）
**操作**：文案末尾必须附完整的数据来源清单。
```
【数据来源】
1. 1062人临床试验数据 — 来源：《NSKSD科学循证文献合辑2026》，发表于Frontiers in Cardiovascular Medicine
2. 浙大RCT研究 — 来源：浙江大学医学院附属第二医院神经内科楼敏教授团队，发表于《卒中与脑血管疾病杂志》
3. 专家建议 — 来源：《纳豆激酶在血管病危险人群应用的专家建议》，北京神经内科学会等3家学术机构
```

### 步骤五：反向质疑（Devil's Advocate）
**操作**：写完后，用批判性视角逐段审问：
- "如果读者去查证这个数据，能查到吗？" → 查不到就删
- "如果竞品拿这篇文章举报，哪里会被抓住？" → 有风险就改
- "如果监管部门看到这句话，会怎么理解？" → 有歧义就重写
- "这个结论是知识库原文的结论，还是AI自己推导的？" → AI推导的不用

### 验证结果记录

每篇文案的合规复审报告中必须包含数据验证部分：

```
### 数据真实性验证（5步验证）
| 数据点 | 文案表述 | 原文出处 | 精确核对 | 交叉验证 | 结果 |
|--------|----------|----------|----------|----------|------|
| 1062人临床 | ✅原文一致 | 科学循证文献 > 第X页 | ✅数字准确 | ✅新闻报道印证 | 通过 |
| ... | | | | | |
```

## 第六章 安全写作模板

### 模板A：引用临床数据（最常用）

```
✅ 安全写法：
"在一项由[医院名称][科室]开展的研究中，[XX]名受试者参与了为期[XX]周的观察。
研究数据显示，[具体指标]呈现[具体变化趋势]。
该研究已发表于[期刊名称]，供学术参考。

温馨提示：本品为食品，不能替代药物。如有健康问题，请咨询专业医生。"

❌ 违规写法：
"临床验证，纳豆激酶能有效降血压，1062名患者证明100%有效！"
```

### 模板B：专家观点引用

```
✅ 安全写法：
"[专家姓名]（[完整职务]）在[具体场合]上指出：
'[直接引用原话]'
以上为专家个人学术观点，仅供参考。"

❌ 违规写法：
"权威专家证明纳豆激酶能治病！浙二医院主任医师都在推荐！"
```

### 模板C：品牌故事/招商软文


## 第七章 招商内容的软转化规范

### 7.1 CTA放置原则
- 所有行动引导**放在文末**，不要在文章中间推销
- 用"了解合作"而非"立即购买"
- 用"想聊聊"而非"立即扫码"
- 用"如果你也在找靠谱的品牌"而非"赶紧加入我们"

### 7.2 招商信息的植入方式
- **好的植入**：通过行业分析、经营方法论、选品逻辑自然引出日生研的优势
- **差的植入**：直接说"日生研招商中，欢迎加盟"
- **原则**：让读者看完有收获感，而不是"又一篇广告"

### 7.3 资料包引流（可选CTA）
- "想了解更多临床数据和分销合作？回复【资料】获取完整资料包"
- 资料包内容：临床研究摘要 + 专家建议原文 + 央媒报道合集 + 分销利润模型 + 产品说明

## 第八章 法规依据汇总（交叉验证来源）

| 法规/规范 | 核心条款 | 与日生研的关系 | 来源 |
|-----------|----------|----------------|------|
| 《广告法》第17条 | 非医疗广告不得涉及疾病治疗 | 纳豆激酶是食品，不得宣传治疗功效 | 全国人大常委会 |
| 《广告法》第18条 | 保健食品广告限制 | 日生研是普通食品，限制更严 | 全国人大常委会 |
| 《广告法》第9条 | 禁止绝对化用语 | 所有宣传材料必须避免 | 全国人大常委会 |
| 《广告法》第4条 | 广告不得虚假误导 | 数据引用必须真实 | 全国人大常委会 |
| 《广告法》第11条 | 引证内容须真实准确标明出处 | 临床数据必须标注来源 | 全国人大常委会 |
| 《食品安全法》第71条 | 食品标签不得涉及疾病预防治疗 | 产品描述必须遵守 | 全国人大常委会 |
| 《食品安全法》第73条 | 食品广告不得涉及疾病预防治疗 | 广告内容必须遵守 | 全国人大常委会 |
| 《食品安全法实施条例》第38条 | 禁止利用学术论文为食品做宣传 | 引用临床数据须客观中立 | 国务院 |
| 四部门《关于规范自媒体医疗科普行为》 | 非医疗资质账号限制 | 健康内容需谨慎定位 | 国家卫健委等四部门 |
| 微信公众平台运营规范 | 健康类内容审核 | 公众号发布必须遵守 | 腾讯 |
| 小红书《广告违禁词规范》 | 绝对化用语、功效承诺禁止 | 小红书发布必须遵守 | 小红书 |
| 抖音《健康内容管理规范》 | 健康科普创作者资质要求 | 抖音发布需确认资质 | 字节跳动 |
