# 2020.3.3-健康时报：浙江中粮联合日本生物驰援武汉蔡甸方舱医院——杜

浙江中粮联合日本生物驰援武汉蔡甸方舱医院


日本生物科学研究所社长东健一郎联合中国战略合作伙伴浙江中粮健康产业公司，日前捐赠价值2000万日元的日本纳豆激酶制品NSKSD，希望为抗击病毒的武汉医护人员提供力所能及的帮助和支援。

作为日本纳豆激酶NSKSD制品的中国总代理浙江中粮公司，董事长袁胜元先生亲赴日本，联手厂家共同捐赠。他说，我们十年如一日，秉持“让老年人不中风、让中年人不心梗”的愿景，专注日本纳豆激酶的科学传播，为国人的健康长寿而努力。疫情当前，能为白衣天使助力，我们理应全力以赴！

武汉蔡甸方舱医院是武汉建设的第15家方舱医院，现有1600张床位，床位数在武汉方舱医院中列前位。大量密集的床位意味着病患的高度密集和大量医护人员的配备，也意味着医护人员的超高工作强度。袁胜元表示，很多医护工作者一站就是十几个小时，而且连续作战，一直处于高压状态，体力透支，心理超负荷，健康受到严重威胁，衷心希望纳豆激酶NSKSD 制品能助他们一臂之力。
