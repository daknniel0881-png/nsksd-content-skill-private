## 一、企业基本信息

- **全称**：日生研生命科学（浙江）有限公司
- **使命**：专注血管健康
- **企业愿景**：让老年人不中风、让中年人不心梗
- **企业理念**：专心、专注、专业
- **成立时间**：2010年起与海外原研方（国际合作研发机构）建立战略合作
- **地位**：NSKSD纳豆激酶中国总代理，品类头部企业
- **董事长**：袁胜元

---

## 二、产品信息

- **产品名称**：NSKSD纳豆激酶
- **生产方**：海外原研方（全球纳豆激酶产业化鼻祖，国际合作研发机构）
- **产品属性**：食品 / 膳食补充剂（**不是药品**）
- **核心成分**：纳豆激酶（nattokinase），由275个氨基酸残基构成的碱性丝氨酸蛋白酶
- **活性单位**：FU（纤溶活性单位）
- **常用剂量**：
  - 1062人临床试验：10800 FU/天
  - 浙大RCT研究：8000 FU/天（每日两次，每次4000FU）
- **安全性**：EFSA（欧洲食品安全局）2017年认证，35岁以上人群可长期服用；不良反应自报率仅1.1%

## 三、临床研究证据链（核心武器库）

### 3.1 最大规模临床试验：1062人

| 项目 | 内容 |
|------|------|
| 规模 | 1062人，迄今纳豆激酶领域最大 |
| 医院 | 中国4家医院（含杭州市第一人民医院等） |
| 方案 | 10800 FU/天，持续12个月 |
| 核心结果 | 颈动脉斑块改善率66.5%-95.4% |
| 发表期刊 | Frontiers in Cardiovascular Medicine（国际期刊） |
| 招商杀伤力 | ⭐⭐⭐⭐⭐ |

### 3.2 浙大RCT研究（全球首个高质量RCT）

| 项目 | 内容 |
|------|------|
| 团队 | 浙江大学医学院附属第二医院神经内科，楼敏教授领衔 |
| 设计 | 随机、双盲、安慰剂对照（RCT）——最高等级证据 |
| 受试者 | 120名无症状性颅内/颈动脉狭窄患者 |
| 剂量 | 8000 FU/天，持续6个月 |
| 核心结果 | 视觉空间功能衰退风险降低约65% |
| 睡眠改善 | 呈现改善趋势（P=0.072） |
| 发表期刊 | 《卒中与脑血管疾病杂志》 |
| 资助 | 国家自然科学基金、中国科学院发展中心 |
| 招商杀伤力 | ⭐⭐⭐⭐⭐ |

### 3.3 杭州市一人体临床

| 项目 | 内容 |
|------|------|
| 核心结果 | 斑块改善率46.7%，中膜增厚改善率42% |
| 发表 | 中国核心期刊《中华养生保健》 |
| 意义 | 中国首例NSKSD纳豆激酶人体临床试验报告 |

### 3.4 其他支撑研究

- **178人RCT**：纳豆激酶+红曲，胆固醇降低0.66mmol/L
- **90人RCT**：收缩压降低7.3mmHg，效果与DASH饮食相当
- **Meta分析**（546人/6项RCT）：可作为高血压有效辅助治疗
- **29000人队列研究**（海外大样本）：习惯食用纳豆与长寿正相关

## 八、市场乱象素材（行业净化者叙事线）

可直接引用的反面案例：
- **成本5元卖300元**：央广网报道，部分厂家纳豆激酶成本仅5元/瓶，售价300+元
- **虚假宣传被罚40万**：普通食品冒充保健食品
- **南京张奶奶案例**：花4000元买的纳豆激酶，成分90%是淀粉
- **直播间骗局**：限时抢购+专家背书+情绪煽动，中老年人是最大受害者
- **假冒进口**：声称海外进口，实为国内小作坊，海外查无踪迹
