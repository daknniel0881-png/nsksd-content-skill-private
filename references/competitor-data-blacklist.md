# 竞品数据黑名单（V10.6）

## 二、黑名单数据指纹（禁止归属日生研/NSKSD）

| 数据点 | 正则模式 | 风险说明 |
|--------|---------|---------|
| **1062人** | `1062\s*[人名例]` | 竞品最核心样本量标签 |
| **4家医院** | `4\s*家医院` | 竞品联合研究描述 |
| **广东双骏** | `广东双骏` | 竞品公司名直接命中 |
| **Frontiers in Cardiovascular Medicine** | `Frontiers\s+in\s+Cardiovascular\s+Medicine` | 竞品论文发表期刊（非日生研发表处） |
| **10800FU×12月** | `10800\s*FU.{0,10}12\s*[月个]` | 竞品特有剂量/疗程组合 |
| **66.5-95.4%改善率** | `66\.5\s*[-~]\s*95\.4\s*%` | 竞品特有效果区间 |

## 三、日生研/NSKSD真实临床数据（对照参考）

### 市一医院研究（主要临床）
- **机构**：杭州市第一人民医院（钱申贤教授团队）
- **样本量**：97例
- **干预**：每日2粒
- **疗程**：90天
- **发表**：《中华养生保健》第42卷第5期（2024）P8-12
- **结果**：斑块改善率46.7%，IMT改善率42.1%，脑血流改善71.4%，异常流速改善87.5%

### 浙大RCT（辅助引用）
- **机构**：浙江大学医学院（楼敏教授团队）
- **样本量**：**120人**（非1062人）
- **干预**：8000FU/天
- **疗程**：6个月
- **发表**：《卒中与脑血管疾病杂志》
- **结果**：认知衰退风险降低65%

## 四、写稿使用规则

1. 写到"浙大RCT"时，**只能用120人**，禁止使用1062人
2. "认知衰退65%"必须配"浙大120人RCT"，不能配1062人
3. 1062人、4家医院、66.5-95.4%、10800FU×12月这四组数据，**一律不得出现在日生研/NSKSD内容里**
4. `fact_auditor.py` 规则E会自动拦截，exit code=1阻断发布
