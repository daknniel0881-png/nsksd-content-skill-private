## 6. 非白名单源使用规则

### 6.1 找不到白名单源时

写占位符，格式为：`[待核实:具体描述需要什么数据]`

示例：
- `[待核实:需要2024年中国高血压患病率官方数据，来源卫健委或中国疾控中心]`
- `[待核实:需要颈动脉斑块在45岁以上人群中的发生率数据，来源权威流行病学报告]`

### 6.2 实在必要时的降级写法

以下两种降级写法允许使用，但每篇文章不得超过 2 处：

- "行业内普遍认为……"（后跟的内容必须是公认事实，不是品牌专有主张）
- "根据内部追踪数据……"（仅适用于品牌自身用户反馈，不能用于科学主张）

### 6.3 绝对禁止的懒惰话术

以下表述无论在什么语境下，一律不得出现：

| 禁用话术 | 问题所在 |
|---------|---------|
| 研究表明 | 无锚点，读者无法核实 |
| 专家认为 | 哪位专家？必须具名 |
| 据统计 | 哪个机构的统计？必须标出 |
| 有报道称 | 哪个媒体？必须具体 |
| 相关数据显示 | "相关"是数据虚化的遮羞布 |
| 业内人士透露 | 无主体，无法核实 |
| 权威人士表示 | 同上 |
| 国外研究 | 哪国？哪个机构？哪篇文献？ |
| 临床证明 | 哪家医院？多少人？什么方案？ |
| 大量研究显示 | 必须说清是几项、什么类型的研究 |
| 权威机构 | 必须具名：是卫健委还是中华医学会？ |
| 科学实验证实 | 必须给出实验来源 |
| 医学界公认 | 医学界没有"公认"，只有共识文件，必须引用具体共识 |

### 6.4 数据引用强制规范（引用时的写法模板）

正确示例：
> "在由杭州市第一人民医院等4家医院开展的1062人临床研究中，受试者每日服用NSKSD纳豆激酶10800 FU，持续12个月，结果显示颈动脉斑块改善率为66.5%-95.4%。该研究已发表于 Frontiers in Cardiovascular Medicine。"

错误示例：
> "大量临床研究表明，纳豆激酶能有效改善颈动脉斑块。"
